== Description ==

QQWorld WooCommerce助手，包含品牌、送货、物流查询等扩展设置。

== Screenshots ==

1. 品牌分类法
2. 送货状态
3. 中国本地化，编辑送货地址
4. 中国本地化，订单页查看收货地址
5. 中国本地化，订单页编辑收货地址


== Installation ==

1. 上传 `qqworld-woocommerce-assistant-lite` 目录到 `/wp-content/plugins/`
2. 在插件页激活 QQWorld Woocommerce Assistant Lite插件
